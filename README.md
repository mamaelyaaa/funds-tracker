# Трекер капитала

Система для отслеживания личного капитала и доходов.

## Быстрый старт

Для локального запуска должно быть установлено:

- **Docker**
- **GNU Make** или **WSL** для работы с make (рекомендуется, но не обязательно) 

### 1. Клонирование репозитория

```bash
git clone https://github.com/mamaelyaaa/funds-tracker.git
cd funds-tracker
```

### 2. Настройка окружения

Создайте файл `.env` на основе `.env.example`. В большинстве случаев значения по умолчанию подходят для локального
запуска.

```shell
cp .env.example .env
```

### 3. Запуск сервисов

В приложении используется `Makefile` как обертка над Docker, Python командами для упрощения работы

```shell
# Сборка образов (для первого запуска и в случае изменения Dockerfile)
make build    

# Запуск всех контейнеров в фоновом режиме
make up       
```

Все доступные команды описаны в `Makefile`. Посмотреть список можно командой:

```shell
make
```

> При отсутствии make использовать основные команды `docker compose`
